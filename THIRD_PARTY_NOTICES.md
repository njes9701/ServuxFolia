# Third-party notices

ServuxFolia implements an interoperable network protocol documented by and
observed in the following open-source projects:

- Servux by masa/sakura-ryoko, LGPL-3.0:
  https://github.com/sakura-ryoko/servux
- Litematica by masa/sakura-ryoko, LGPL-3.0:
  https://github.com/sakura-ryoko/litematica
- LitematicaFolia by Ekaii, MIT:
  https://forgejo.ekaii.fr/admin_ekaii/litematica-folia-ekaii

Channel names, packet identifiers and wire layouts are used for
interoperability. No Fabric runtime or Servux binary is included. Any source
later adapted from an upstream implementation must retain its original notice
in the corresponding file.
